# D3ObyTCPServer

A simple TCP server made into ruby gem.

## Installation

Add this line to your application's Gemfile:

    gem 'D3ObyTCPServer'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install D3ObyTCPServer
